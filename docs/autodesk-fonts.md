# AutoCAD 原版 SHX 字体清单（初始安装释放）

[← 返回主页](../README.md#📜-字体来源声明)

## 说明

- 该清单用于区分“原版基线字体”与“后续额外补充字体”。
- 若你本地 `Fonts` 目录与此清单不一致，通常是因为安装了行业插件、补丁包或手动增删字体。
- 文件名保留原始大小写/命名形式（含特殊条目，如 `@extfont2.shx`、带空格文件名）。
